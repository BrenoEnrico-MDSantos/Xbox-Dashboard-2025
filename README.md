# Xbox-Dashboard-2025
Dashboard criado para o desafio do Bootcamp DIO Santander Ciência de Dados 2025.
* Apresenta dados fictícios relativos a um período de vendas, com enfoque em assinantes de diferentes planos e escolhas.
* A visualização é dinâmica e faz uso de *Campos Calculados*, *Ação* de _filter reset_ e um *Parâmetro* para troca de gráfico principal.
